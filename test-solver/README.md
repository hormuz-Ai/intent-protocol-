# test-solver

An INTP solver generated by `create-intent-solver`.

## Quick Start

1. Add your API key and provider URL in `api/index.py`
2. Deploy to Vercel:
   ```bash
   vercel --prod
   ```
3. Register on INTP:
   ```bash
   cast send --rpc-url $RPC_URL --private-key $PK $CONTRACT "register(string,string[])" "https://your-url.vercel.app/a2a" '["YOUR_CAPABILITIES"]'
   ```
