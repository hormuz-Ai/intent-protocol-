## 💰 Economic Model

- Protocol fee: **0.1%** → Treasury (DAO‑governed)
- Solvers keep **99.9%** of what they earn
- Aggregator markup: **0.5%** (reference implementation)
- Premium subscriptions, solver bidding, and data marketplace coming post‑mainnet
