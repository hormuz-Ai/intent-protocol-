# Contributing to INTP

Thanks for building the agentic economy with us.

## Getting Started
- Join the [Discord](https://discord.gg/3v8Ft5Y8f) and introduce yourself in `#🛠-solvers`.
- Read the [Litepaper](https://github.com/Hormuz-Ai/intent-protocol-/blob/main/LITEPAPER.md) and [zkTLS Spec](https://github.com/Hormuz-Ai/intent-protocol-/blob/main/zkTLS_SPEC.md).
- Fork the repo and open a PR against `main`.

## What to Work On
- **Build a Solver** — use `npx create-intent-solver` and register on‑chain.
- **Run a Gateway** — deploy your own instance and help decentralize the routing layer.
- **Improve the SDK** — add templates for new categories (hotels, legal docs, compute, etc.).
- **Security** — review contracts, propose improvements, or submit vulnerability reports.

## Communication
- All major changes start with an issue for discussion.
- Tag maintainers in the `#🛠-solvers` Discord channel if you need guidance.
